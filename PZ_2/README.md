# Практична робота №2: ОСНОВИ ВЕБ-ПРОГРАМУВАННЯ

**Тема:** JavaScript. Динамічний контент
**Варіант 9:** Підстанція 330/110 кВ

## Про проєкт
Цей проєкт являє собою інтерактивну веб-сторінку для моніторингу ключових параметрів енергетичного об'єкта (Підстанції 330/110 кВ) в режимі реального часу. Мета роботи — демонстрація навичок роботи з DOM-деревом, подіями, таймерами та зовнішніми API за допомогою чистого JavaScript.

## Основний функціонал
* **Динамічна генерація даних:** випадкова генерація показників напруги, струму та частоти в межах заданих діапазонів
* **Кольорова індикація станів:** автоматичне визначення та підсвічування статусу параметра (норма — зелений, попередження — жовтий, критично — червоний).
* **Керування оновленням:** можливість оновлювати дані вручну або увімкнути автоматичний таймер з інтервалом у 3 секунди.
* **Візуалізація (Chart.js):** побудова динамічного лінійного графіка зміни напруги 330 кВ із відображенням статичних меж норми.
* **Локальне сховище:** збереження історії вимірювань у пам'яті браузера (`localStorage`), що дозволяє не втрачати дані графіка при перезавантаженні сторінки.

## Технологічний стек
* **HTML5:** семантична розмітка сторінки
* **CSS3:** стилізація, адаптивний дизайн за допомогою CSS Grid
* **JavaScript :** клієнтська логіка, генерація даних, маніпуляції з DOM
* **Chart.js:** зовнішня бібліотека для малювання графіків
