# Практична робота №1. Розробка веб-сайту енергетичного об'єкта

Репозиторій містить вихідний код до практичної роботи з дисципліни "Основи веб-програмування".
Тема: HTML та CSS. Розробка інформаційного веб-сайту енергетичного об'єкта.

## Інформація про варіант

**Варіант:** 9
**Об'єкт:** Підстанція глибокого вводу ПГВ-1
**Технічні характеристики:**
- Тип: Трансформаторна підстанція
- Потужність: 40 МВА
- Напруга: 110/10 кВ
- Місцезнаходження: м. Одеса, вул. Промислова

## Використані технології

- **HTML5**: Семантична структура сторінки (main, section, nav, footer).
- **CSS3**: Власні стилі, використання Flexbox та анімацій (@keyframes).
- **Bootstrap 5**: Адаптивна верстка, використання системи сіток (Grid System) та готових компонентів.

## Структура проєкту

- `index.html` — основний файл розмітки.
- `css/styles.css` — файл каскадних таблиць стилів.
- `images/` — папка з графічними матеріалами (схема, фото об'єкта).
